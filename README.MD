# EmojiHub - Каталог эмодзи с избранным

## Краткое описание
EmojiHub - это веб-приложение для работы с эмодзи, предоставляющее:
- Просмотр каталога с поиском, фильтрацией и сортировкой
- Возможность добавления в избранное
- Управление персональной коллекцией
- Аутентификацию пользователей
- Адаптивный интерфейс с выбором стиля отображения

## Установка и запуск
1. Клонировать репозиторий:
```bash
git clone https://github.com/Zhangeldi123/NfactorialIncubator
cd emojihub
Установить зависимости:

bash
pip install -r requirements.txt
Создать файл .env в корне проекта:

ini
SECRET_KEY=your-secret-key
DEBUG=True
DATABASE_URL=postgres://user:password@localhost/emojihub
Применить миграции:

bash
python manage.py migrate
Собрать статические файлы:

bash
python manage.py collectstatic
Запустить сервер:

bash
python manage.py runserver
Для продакшн-среды используйте:

bash
gunicorn --workers 3 your_project.wsgi
